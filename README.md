# 🛍️ GH Vendas — Site Oficial

Site institucional da **GH Vendas**, desenvolvido para apresentar serviços de vendas online, parceiros confiáveis e canais de contato com clientes.

O projeto foi criado com foco em design moderno, navegação simples e experiência do usuário.

---

## 🚀 Funcionalidades

✔ Página inicial com apresentação do negócio  
✔ Página sobre a empresa  
✔ Página de contato com acesso direto ao WhatsApp  
✔ Página com grupos e redes sociais  
✔ Página com empresas parceiras  
✔ Design moderno com animações suaves  
✔ Botão de voltar ao topo automático  
✔ Layout responsivo  

---

## 📄 Estrutura do Site

- `index.html` → Página inicial  
- `sobre.html` → Informações sobre a empresa  
- `contato.html` → Formas de contato  
- `grupos.html` → Comunidades e redes sociais  
- `parceiros.html` → Empresas parceiras  
- `style.css` → Estilos e design visual  
- `script.js` → Interações e animações  

---

## 🧑‍💻 Tecnologias Utilizadas

- HTML5  
- CSS3 (Glassmorphism + animações modernas)  
- JavaScript (DOM, animações, scroll effects)  

---

## 🎯 Objetivo do Projeto

Apresentar a GH Vendas como uma empresa confiável no mercado digital, destacando:

- vendas online  
- atendimento direto  
- empresas parceiras  
- canais de comunicação com clientes  

---

## ▶ Como Executar o Projeto

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

Ou publique online usando GitHub Pages.

---

## 🌐 Publicação Online (GitHub Pages)

1. Envie o projeto para um repositório no GitHub  
2. Vá em **Settings → Pages**  
3. Selecione a branch principal  
4. O site ficará disponível com um link público
